üöÄ Quick Setup Guide

1 Install Composer (if not installed)

Composer is required to install Laravel dependencies.

  • Download from: https://getcomposer.org/download/
  • Run Composer-Setup.exe
  • Select your XAMPP PHP: C:\xampp\php\php.exe
  • Complete the installation

Verify installation:

composer --version

2 Install Laravel Framework

Open PowerShell or Command Prompt in this folder and run:

cd "c:\xampp1\htdocs\new 360 Project"
composer create-project laravel/laravel temp "11.*"
Move-Item -Path temp\* -Destination . -Force
Remove-Item temp -Recurse -Force

OR simply double-click: start.bat in this folder

3 Configure Environment

Copy and configure the environment file:

Copy-Item .env.example .env
php artisan key:generate

Edit .env file and set:

DB_DATABASE=360winestate
DB_USERNAME=root
DB_PASSWORD=

MAIL_MAILER=log

4 Create Database

  • Start XAMPP Control Panel
  • Start Apache and MySQL
  • Open: http://localhost/phpmyadmin
  • Click "New" ‚Üí Database name: 360winestate
  • Click "Create"

5 Run Migrations & Seed Data

php artisan migrate
php artisan db:seed --class=UserSeeder

6 Start the Server

php artisan serve

Then open in browser:

Open Application Register Page Login Page
